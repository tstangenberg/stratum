# US-0045: Enable GitHub Sponsors

**As a** supporter of Stratum  
**I want** a way to financially support the project  
**So that** I can contribute even if I'm not a developer

## Context

Enabled at OSS launch. Low-friction, no expectation of income at this stage. Funds go toward infrastructure costs. Signals intent and sustainability to early adopters.

## Acceptance Criteria

- [ ] GitHub Sponsors is enabled for Thorben's GitHub account
- [ ] Sponsorship tiers are defined (at minimum: a $5 "supporter" tier)
- [ ] `.github/FUNDING.yml` is added to the repo pointing to the Sponsors profile

## E2E Tests

Manual verification: confirm Sponsors button appears on the repo and the FUNDING.yml file exists in the repository.
