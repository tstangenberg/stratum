# US-0043: Set up public roadmap

**As a** potential adopter or contributor  
**I want** to see what's planned for Stratum  
**So that** I can evaluate fit, contribute where it matters, and trust the project is actively maintained

## Context

A GitHub Projects board made public at OSS launch. Columns map to story status. Milestones map to story batches.

## Acceptance Criteria

- [ ] GitHub Projects board created with columns: Backlog → Ready → In Progress → Done
- [ ] Board is set to public visibility
- [ ] All existing stories in `docs/stories/ready/` are added to the board

## E2E Tests

Manual verification: open the board as an unauthenticated user and confirm it is visible and the community launch stories appear.
